## Instalando e Rodando

Clone o repositório e entre no diretório

```bash
$ git clone https://github.com/ZeEdu/weather-challenge.git
$ cd weather-challenge

```

Instale as dependências com

```bash
$ yarn install
```

ou

```bash
$ npm install
```
